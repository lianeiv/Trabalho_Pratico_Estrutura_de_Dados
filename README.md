# Trabalho_Pratico_Estrutura_de_Dados
Sistema em C para gerenciamento de Clientes, Produtos e Carrinho de Compras, utilizando listas simplesmente encadeadas e alocação dinâmica de memória. O projeto implementa operações completas de CRUD via terminal, sem uso de variáveis globais e com manipulação das listas por passagem de parâmetros.  Este repositório corresponde à reescrita individual do trabalho prático originalmente desenvolvido em grupo, com o objetivo de aprofundar o entendimento sobre listas encadeadas, organização de código em C e correção de falhas identificadas na versão anterior.
